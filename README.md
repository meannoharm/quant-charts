## 同步源仓库后续提交

添加上游仓库

```bash
git remote add upstream https://github.com/tradingview/lightweight-charts
```

拉取上游变动

```base
git fetch upstream
```

合并

```bash
git merge upstream/master
```

更新自己远端仓库分支

```bash
git push origin master
```
